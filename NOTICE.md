# Third-Party Notices

This repository contains original project code plus cleaned experiment summaries and figures.

## LongBench

The project references and evaluates LongBench-style tasks. The bundled notice file `third_party/LongBench_LICENSE.txt` is the MIT license from the LongBench repository material used during the original course project.

## LLMLingua / LongLLMLingua

The optional `llmlingua` compressor is loaded lazily through the external `llmlingua` Python package. This repository does not vendor the package or model weights.

## Datasets

The cleaned release does not redistribute large raw datasets. Users should download NaturalQuestions, Lost in the Middle, and LongBench resources from their official or upstream project locations and follow their individual licenses and terms.

## API Services

Answer generation scripts can call a DeepSeek-compatible OpenAI SDK endpoint when users provide their own API credentials. No API key is included in this repository.
